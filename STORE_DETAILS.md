## 🌐 English Listing Details

### 1. Agent Name (Title)
ProjectGuard AI | Code Splitter & Drift Guard

### 2. Subtitle (One-line Summary)
An interactive AST architectural governance tool to enforce strict module boundaries and prevent AI-generative code drift.

### 3. Details (Long Description)
Have you noticed that as your AI programming sessions grow longer, the code begins to "drift" or "degrade"? Large Language Models (LLMs) often suffer from polymorphic memory decay during extended developer workflows, preferring to stack thousands of lines of new logic into a single monolithic file (like `App.tsx`) rather than performing clean, physical folder modularization.

**ProjectGuard AI** solves this critical problem. Operating as a lightweight, zero-dependency, and highly responsive static auditing engine right inside your browser, it acts as a sentinel for your codebase. It parses your TSX/JS code on the fly to enforce strict modular contracts (`ARCHITECTURE.json`), protecting your team's agreed-upon structural boundaries from spaghetti regression.

*Available on the Capafy platform.*

---

### 🚀 Key Capabilities
* **Declarative Contract Specification**: Define file-length caps, cyclic import rules, and allowed dependency matrices easily via a clean, human-readable JSON contract template.
* **Dual Parsing Engine**: Employs an ultra-fast local Abstract Syntax Tree (AST) analyzer to perform rigid interface verification (`.types.ts` / `.utils.ts`) alongside elastic workspace sensing (Next.js, Vue modules) in real-time.
* **Code Splitting & Refactoring Directives**: Instantly maps imports, detects circular coupling vectors, and outputs mathematically step-by-step physical code splitting advice.
* **Simulation Lab**: Allows developers to stress-test their architecture by simulating structural drift, coupling bloat, and architectural degradation in real-time.
* **Active Snapshot Recovery**: Automatically saves clean snapshots, allowing developers to roll back non-conforming edits with zero lost effort.

### 🎯 Use Cases
* **AI-Guided Refactoring Guardrail**: Inject strict architectural rules directly into downstream AI prompts to enforce modular clean-code standards during continuous generation rounds.

### 🚀 核心功能特点
* **声明式架构契约规范**：提供极简、易读的 JSON 契约设定，包括单个文件行数上限、禁止循环依赖、限定模块间引用白名单（allowedDependencies）等。
* **刚性与弹性双轨解析**：静态解析器可将通用纯逻辑契约规范绑定至 `.types.ts` 和 `.utils.ts`（刚性规则），同时能够敏锐识别 Next.js, Vue 等特有目录指纹（弹性调整），实现智能格式兼容。
* **实时 AST 引流与解耦计算**：在毫秒级内绘制模块依赖图谱，分析引用关系，并精准输出无损物理拆分的三步重构指示。
* **架构退化实验室**：提供模拟环境，允许开发者一键注入“高耦合代码”、“依赖崩溃”、“巨型单体文件”等退化指标，实时观察系统指标受损与警戒过程。
* **版本快照历史与安全回退**：实时记录每个 100% 达标的架构状态，在发生代码退化时提供一键安全物理还原，零研发心智负担。
